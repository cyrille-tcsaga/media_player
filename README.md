# Lecteur Média PyQt6

Lecteur audio/vidéo local, mono-utilisateur, construit avec Python et PyQt6/QtMultimedia.

## Prérequis

- Python 3.11+ (testé avec Python 3.12)

## Installation

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
